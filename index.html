fwoosh

A lightening quick MDX static website generator

A 💀 simple static website generator

  • ⚡️
    Lightening quick builds powered by esbuild
  • 🗄
    File System Based Routing: New pages are as easy as adding a file
  • 📝
    Pages can be authored in modern JavaScript, TypeScript, or MDX
  • 🎁
    Build wrappers for you pages using Layouts
  • 🔌
    Use plugins to build upon fwoosh, add your own assets, layouts, and more

Getting Started

First you need to install fwoosh.

npm i --save-dev fwoosh

# Or when using yarn
yarn add -D fwoosh
npm i --save-dev fwoosh

# Or when using yarn
yarn add -D fwoosh

Once installed you need to define the place where you are going to store your pages. The default is the docs/ folder, but this can be customized using the dir option.

Usage

The fwoosh cli comes with a few easy to use commands:

  • clean: Delete all of fwoosh's build and output files
  • build: Build your static website
  • dev: Start the fwoosh development server

To get more information about each command use the --help flag.

fwoosh --help

# Or get help for a specific command
fwoosh build --help
fwoosh --help

# Or get help for a specific command
fwoosh build --help

Deploying

To build your website for deploying use fwoosh build. This creates a set of simple HTML and JS files that represent your static website. All you need to do is upload it to a server somewhere to be hosted. This can be a GitHub pages website, vercel, netlify, an s3 bucket, really anywhere you want 🎉
